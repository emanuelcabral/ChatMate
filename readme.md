# 🤖 ChatMate

**ChatMate** es un chatbot inteligente y totalmente gratuito que utiliza **modelos de lenguaje abiertos**, junto con **LangChain** y **LangGraph**, para mantener conversaciones naturales y útiles.  
Está diseñado para ser **fácil de usar**, con **memoria persistente** y una interfaz web **atractiva y amigable** construida con **Streamlit**.

---

## ✨ Características principales

- 💬 Conversación en tiempo real con **memoria corta y larga**  
- 🗄️ Memoria persistente en `data/memory.json`  
- 🌈 Interfaz visual amigable con **emojis y Markdown**  
- ⚡ Totalmente gratuito y **listo para usar**  
- ☁️ Desplegable fácilmente en **Streamlit Cloud**  

---

## 🛠 Tecnologías utilizadas

| Componente | Tecnología / Librería |
|------------|---------------------|
| Lenguaje | Python 3.10+ |
| LLM | Hugging Face Transformers (gratuito) |
| Workflows | LangChain, LangGraph |
| Interfaz web | Streamlit |
| Memoria | JSON (`data/memory.json`) |
| Persistencia | Local |
